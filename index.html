Topic Overview

Boolean Algebra

The mathematical foundation of digital circuits and computer logic. It deals entirely with the binary number system, operating strictly on states of true (1) and false (0).

01. Origins & Purpose

The foundational history of the algebra that deals with the binary number system.

George Boole (1815-1864)

English mathematician who developed the algebra to:

  • Simplify logical representation
  • Manipulate propositional logic mathematically

Claude E. Shannon (1938)

Proposed using Boolean algebra in the design of relay switching circuits.

  • An economical & straightforward approach
  • Extensively used in modern electronic computer circuits

04. Boolean Functions

A Boolean function is an expression formed with binary variables, operators, parentheses, and an equal sign. Its resolved value yields either 0 or 1.

Algebraic Expression

W = x + y · z

Alt: W = f(x, y, z) = x + y · z

  • Expression: The Right Hand Side (RHS) of the equation.
  • Literals: The individual symbols representing variables (e.g., x, y, z).

Truth Table Architecture

A function can be represented by multiple algebraic expressions, but it possesses only one unique Truth Table.

  • Row Calculation The number of rows equals 2n, where n is the number of literals.
  • Binary Counting The combinations are obtained by counting in binary from 0 to 2n - 1.

05. Minimization of Functions

Minimization deals with the reduction in the number of literals and the number of terms within an expression.

Objective: Achieve equal but simpler expressions to realize the function with fewer electronic components, resulting in a cheaper circuit.

Original Function (3 Terms, 3 Literals)

F1 = x·y·z + x·y·z + x·y

Optimized

Minimized Function

F2 = x·y + x·z

Algebraic Proof (F1 ≡ F2)

F1 = x·z·(y + y) + x·y Factor out xz
= x·z·(1) + x·y Complement Law
= x·z + x·y Identity Law
= x·y + x·z Commutative = F2

06. Logic Gates

Logic gates are the physical hardware implementations of Boolean operators. They form the building blocks of all digital circuits.

AND

AND Gate

z = x · y

Output is 1 ONLY if both x and y are 1. Multiplicative logic.

OR

OR Gate

z = x + y

Output is 1 if EITHER x or y is 1. Additive logic.

NOT

NOT Gate (Inverter)

z = x

Inverts the input. If x is 1, output is 0. Indicated by the small bubble.

07. Standard Forms (SOP & POS)

To implement Boolean functions systematically, they are often converted into one of two canonical forms using Minterms or Maxterms.

Sum of Products (SOP)

An expression containing purely AND terms, summed together with ORs. Represents the rows in a truth table where output is 1.

Structure

F = (x·y) + (x·z) + (y·z)

  • Combines Minterms.
  • Hardware: 2-level AND-OR circuit.

Product of Sums (POS)

An expression containing purely OR terms, multiplied together with ANDs. Represents the rows in a truth table where output is 0.

Structure

F = (x+y) · (x+z) · (y+z)

  • Combines Maxterms.
  • Hardware: 2-level OR-AND circuit.
